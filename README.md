# Azure VM Static Website Hosting

## Overview
This project demonstrates hosting a static website on an Azure Virtual Machine using a Linux server and web server configuration.

## What I did
- Created a Linux Virtual Machine on Microsoft Azure
- Configured networking and security rules (NSG)
- Connected to the VM using SSH
- Installed and configured a web server
- Deployed a static website
- Verified access via public IP
